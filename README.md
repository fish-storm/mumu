# mumu
# 木木旅行网 毕设项目

木木旅行是一个旅行信息展示平台 主要有三个系统功能：
- 酒店信息查看和检索，各平台比价
- 游记撰写和查看
- 完善的用户系统和个人中心

## 技术栈
- java　SSM框架组
- Htmlunit 仿浏览器爬虫库
